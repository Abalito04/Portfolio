## ⚙️ Run Locally

No installation needed — it's pure static HTML.

```bash
# Clone the repository
git clone https://github.com/Abalito04/Portfolio.git

# Open in your browser
cd Portfolio
open index.html   # macOS
# or just double-click index.html on Windows/Linux
```
